# Rust TCP chat
## Инструкция для запуска
### 1. Запустите сервер
```bash
cargo run -- server
```

### 2. Добавьте первого клиента. Параметр `name` может быть произвольным.
```bash
cargo run -- client --name romeo
```

### 3. Добавьте 2-го клиента. Параметр `name` может быть произвольным.
```bash
cargo run -- client --name julliet
```
